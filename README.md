# 📚중고책 거래 플랫폼 & 관리자 페이지

## 📌 프로젝트 소개
이번 프로젝트는 **중고책 거래 서비스(Book-Main)** 와 **운영 관리(Admin Manager)** 를 따로 만든 프로젝트입니다  
사용자는 중고책을 등록·검색·거래할 수 있으며, 관리자는 게시물/회원/거래 내역을 관리할 수 있도록 구성되어 있습니다.

---

## 📂 프로젝트 구조

/
├── book-main/ # 사용자용 중고책 사이트
│ ├── book-main/ # 실제 사용자 서비스 코드
│
├── bookManager-main/ # 관리자 페이지
│ ├── bookManager-main/ # 관리자 기능 구현 코드
│
└── .gitignore


---

## 🛒 1. 사용자 페이지(Book-Main)

### ✨ 주요 기능
- 📗 중고책 등록 및 수정  
- 🔍 제목 기반 도서 검색  
- 🛒 장바구니 / 구매 기능  
- 👤 회원가입 및 로그인  
- 💬 리뷰 및 평점  
- 📦 배송 진행 상태 확인  

---

## 🛠 2. 관리자 페이지(BookManager-main)

### ✨ 주요 기능
- 👥 회원 관리
- 📚 도서·게시물 입고 관리, 관리자 직급별 수행기능
- ⚙️ 서비스 환경 설정

---

## 💻 기술 스택
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Spring / Java
- **IDE:** STS4, VScode  
- **DB:** MySQL  
- **Build:** Maven
- **Version Control:** Git & GitHub 
- **오류 및 코드검증:** GPT
